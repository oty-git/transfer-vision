Methodology — Pro-BTA + API + GIP
01
Positional Profile + GIP
Role is defined first: e.g. Ball-playing CB or Box-to-box MF. Then Game Intensity Profile (GIP) is assessed from video — decision time available, opponent pressure density — scored 0–1000. GIP becomes the context multiplier for all subsequent scores.
02
Pro-BTA Audit (0–1000)
Four blocks scored independently:

BKE 2.0 — Biomechanics: turns, first step, balance under pressure.
TPA 2.0 — Technique: release time (RTI), pass accuracy, weak foot.
CPS 2.0 — Cognition: scanning frequency, decision latency.
CTI 2.0 — Context: impact on game, adjusted for GIP intensity.
03
TVS 2.0 Calculation + API
Base Score = (BKE + TPA + CPS + CTI) / 4

Three multipliers applied:
KGIP: Low ×0.7 · Med ×1.0 · High ×1.2
Kleague: Africa ×0.6 · Top-5 EU ×1.2
Kage: 17–19 ×1.5 · 20–22 ×1.3 · 23–25 ×1.1

API (Age & Potential Index) generates current TVS + projected pTVS in 3 years.
